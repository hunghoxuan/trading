# Session Playbook

Use after reading `rules/handoff.md`.

Refresh:
1. Read `.agents/README.md`.
2. Read `.agents/rules.md`.
3. Read `.agents/worklog.md`.
4. Read active sprint/bugs.

Handoff:
1. Update `.agents/worklog.md`.
2. Update `.agents/sync/MAILBOX.md` only for agent relay.
3. Record pending work and blockers.

Rewind:
- Prefer `git revert`.
- Avoid destructive reset unless user explicitly approves.
