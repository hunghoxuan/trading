# Skill: Capture System Discovery

## Trigger
Identify when a deep investigation has revealed the inner workings of a feature.

## Steps
1. **Locate Feature**: 
   - Check [.agents/tickets/feature_tracker.md] to see if the feature already exists.
   - If not, add it to the tracker and create a new doc in [.agents/features/2-done/].
2. **Update Feature Doc**:
   - Instead of creating a new file, update the existing feature doc in `2-done/`.
   - Maintain the "User Flow" and "Key Capabilities" sections.
   - Add new findings to "Technical Details".
3. **Log Action**:
   - Record the documentation update in [.agents/tickets/4-done/_archive-worklog.md] as a "Knowledge Sync".
4. **Report**: Inform the user that the Feature Source-of-Truth has been updated.
