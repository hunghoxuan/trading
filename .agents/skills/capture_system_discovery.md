# Skill: Capture System Discovery

## Trigger
Identify when a deep investigation (using grep, find, or multi-file reading) has revealed the inner workings of a feature.

## Steps
1. **Summarize**: Create a concise summary of the discovery (Logic flow, File paths, Event types).
2. **Locate Target**: Open `.agents/tickets/4-done/_archive-worklog.md`.
3. **Format Entry**:
   - Date & Time
   - Discovery Topic
   - Key Findings (Bullet points)
   - Code References (File paths + Lines)
4. **Apply**: Write the update to the file.
5. **Report**: Inform the user that the knowledge has been persisted.
