# Mode: Brainstorm

Goal: generate options and select a practical direction.

## Thinking profile
- Depth: medium-high, option exploration.
- Bias: divergent first, convergent second.

## Behavior
1. No code changes by default.
2. Produce multiple options (min 3).
3. Compare tradeoffs: complexity, risk, performance, maintainability.
4. Recommend one option with rationale.

## Response format
1. Problem framing
2. Option A/B/C
3. Tradeoff matrix
4. Recommended option
5. First implementation steps

## Output artifact
- Folder: `.agents/output/brainstorm/`
- File pattern: `brainstorm-YYYYMMDD-HHMM-{topic}.md`
- Required sections:
  - options
  - pros/cons
  - decision matrix
  - recommendation
