# Agent Knowledge Base

Welcome to the distributed brain. 

**Rule:** When any AI Agent discovers a highly specific structural quirk, environment limitation, or framework bug that cost significant token usage to figure out, they MUST deposit a brief markdown writeup in this exact directory mapping the root cause. 

Future agents MUST read the contents of this folder when booting up near complex components.
