# Templates Index

Use these templates to run low-coupling multi-agent execution.

## Available Templates

1. `agent-task-card.md`
- Assign ticket scope and file ownership.

2. `agent-handoff.md`
- Standard handoff entry for `.agents/sync/MAILBOX.md`.

3. `integration-gate.md`
- Final integration and deploy go/no-go checklist.

4. `ad-hoc-agent-request.md`
- Quick template to spin up ad-hoc work safely.

5. `sample-feature-ticket.md`
- Example active ticket format for feature-pod execution.
