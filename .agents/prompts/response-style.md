# Response Style Controls

Use this file to tell the agent how to answer.

## Quick Controls
- `Structure`: `minimal` | `standard` | `detailed`
- `Tone`: `direct` | `neutral` | `strict`
- `Depth`: `fast` | `normal` | `deep`
- `Planning`: `none` | `short-plan` | `full-plan`
- `Confirmation`: `ask-only-when-blocked` | `ask-before-major-change`
- `Language`: `vi` | `en` | `mixed`

## Example Prompt
```text
Use response style:
- Structure: standard
- Tone: direct
- Depth: normal
- Planning: short-plan
- Confirmation: ask-only-when-blocked
- Language: mixed
```
