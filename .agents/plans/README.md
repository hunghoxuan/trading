# Plans Index

Read:

1. `sprint.md`
2. `bugs.md`
3. `backlog.md` when planning future work
4. `changelog.md` when checking history

Rules:
- Sprint = active work.
- Bugs = active defects.
- Backlog = future/parked work.
- Changelog = completed history.
- Respect ownership tags.
- Mark DOING/DONE when work starts/finishes.
